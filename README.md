# perovskite_chain
Skoltech BCEI 2026 project

# Guide 

Как с этим всем работать:

1) Файл esp_code содержит программку для ESP12F, с которой мы работали. Её можно передать на плату после подключения стадартным образом через Arduino IDE. Сама программа читает из ADC порта показания с батареи и отправляет вольтаж на сервер,

2) Чтобы отпрака работала корректно нужно подключить и компьтер с сервером, и вашу плату к общему WiFi. Подключение к общественному WiFi со входом после ввода пароля неприменимо, ESP12F не такая умная,

3) Файл server.js содержит код тривиальной странички, получающей запросы и выводящей стэйт на экран. Установка и запуск стандартные для любого node.js сервера через npm.
